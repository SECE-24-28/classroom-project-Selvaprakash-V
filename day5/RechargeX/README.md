# RechargeX

Fast, mobile-friendly recharge experience with smart plan discovery and confirmation flows.

## Live Demo
- https://singular-starship-a07382.netlify.app/

## Features
- Responsive Tailwind UI with hero, features, and reviews sections
- Plan discovery with operator filter, price slider, and mock API fallback
- Recharge flow with validation, selection grid, and confirmation modal
- Simple register and login forms with inline validation

## Getting Started
1. Clone this repository.
2. Open `index.html` in your browser (or serve the folder with any static server).
3. Plans load from a public mock API and gracefully fall back to built-in sample plans.
